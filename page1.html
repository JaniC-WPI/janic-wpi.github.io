Vision-based control

Skeleton-based Adaptive Visual Servoing

Existing approaches in vision-based control literature utilize information about the robot's end effector pose in the image to close the control loop. Consequentially, these approaches only assert control in the robot's task space. My skeleton-based approach enables us to control the entire shape of redundant robots using purely morphological information.

Keypoints-Based Adaptive Visual Servoing for Control of Robotic Manipulators in Configuration Space

This project introduces an open-source automated data collection pipeline that generates keypoint annotations for robotic arms using camera parameters and forward kinematics. The trained real-time keypoint detector enables purely vision-based, model-free control without relying on encoders or proprioceptive sensors, achieving adaptive visual servoing using only natural image features.

Utilizing Inpainting for Training Keypoint Detection Algorithms Towards Markerless Visual Servoing

This project introduces a novel strategy for training keypoint detection models to track natural features on robotic manipulators by leveraging inpainting techniques to remove fiducial markers after initial labeling. The trained real-time keypoint detector enables fully model-free adaptive visual servoing, using only natural robot features for control without reliance on external markers or prior robot knowledge.

Vision-only Motion Planning

Image-Based Roadmaps for Vision-Only Planning and Control of Robotic Manipulators

This project presents a novel motion planning framework for robotic manipulators that computes collision-free paths directly in image space. By constructing a roadmap from visual features and using learned or predefined distance metrics, the approach allows robotic manipulators to perform vision-only path planning and adaptive control without relying on explicit robot models or onboard proprioceptive sensors.
